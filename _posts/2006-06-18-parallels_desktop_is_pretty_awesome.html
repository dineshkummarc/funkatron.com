Running Zend Studio 5.2 in WinXP within Parallels Desktop on my 20” iMac Core Duo (1.5GB RAM).  It runs pretty nicely, and since it’s a crapshoot getting the ZDE installed on an Intel Mac, quite handy.

Parallels Desktop has gotten better since the last time I tried it.  Seems more stable, and has added some handy features, like automatically adding a network share on your OS X host to allow you to map it as a drive in Windows.  CD/DVD handling seems to work better now too, although I have only tried reading, not writing disks.

I really highly recommend an Intel Mac for anyone who’s interested in running OS X and Windows (even for midlevel gaming).  I’ve been extremely happy with this purchase.  If Xen and/or VMWare come out with host apps for their virtualization systems, it’s gonna be utterly killer.
