So this new blogging tool from Microsoft, Windows Live Writer, is pretty nice.  It’s the first Windows blogging client that works with LJ that didn’t make me want to stab myself in the eye.  It also apparently works with Blogger, TypePad, WordPress, Movable Type, and other stuff that uses the Metaweblog API or the Movable Type API.

Microsoft’s embracement of the blogging community seems to have inspired much better UI design and use of open stadards.  I’m very pleased.

The image insertion stuff is the bomb.  It auto-generates a thumbnail with a neat little dropshadow, and you can apply watermark text and basic effects (sepia, b&w, sharpen, et al), and do more operations.

Hint: when you browse for a picture it only filters for .gif and .jpg, but you can change the file type to All Files and use PNGs.
